# Despliegue-de-CMS-en-arquitectura-en-3-capas.
Se desea desplegar un CMS Wordpress en alta disponibilidad y escalable en AWS. Para ello, se utilizará a en tres capas  como la siguiente:  Capa 1: Capa pública. Balanceador de carga. Capa 2: Capa privada. Servidores de Backend + NFS. Capa 3: Capa privada. Servidor de BBDD.
